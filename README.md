# antiinput

Research workspace for building and testing anti-input-lag hypotheses for competitive shooters.

## Current focus

- objective testing for route quality, client feel, and duel consistency
- rollback of questionable Windows gaming tweaks
- analysis of developer-side compensation systems such as interpolation, buffering, prediction, and rewind
